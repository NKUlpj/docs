[0188]. best-time-to-buy-and-sell-stock-iv ​

难度： 困难

标签： [数组]、[动态规划]

1. Problem ​

给你一个整数数组 prices 和一个整数 k ，其中 prices[i] 是某支给定的股票在第 i 天的价格。

设计一个算法来计算你所能获取的最大利润。你最多可以完成 k 笔交易。也就是说，你最多可以买 k 次，卖 k 次。

注意：你不能同时参与多笔交易（你必须在再次购买前出售掉之前的股票）。

https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-iv/description/

2. Example ​

示例 1：

输入：k = 2, prices = [2,4,1]
输出：2
解释：在第 1 天 (股票价格 = 2) 的时候买入，在第 2 天 (股票价格 = 4) 的时候卖出，这笔交易所能获得利润 = 4-2 = 2 。
输入：k = 2, prices = [2,4,1]
输出：2
解释：在第 1 天 (股票价格 = 2) 的时候买入，在第 2 天 (股票价格 = 4) 的时候卖出，这笔交易所能获得利润 = 4-2 = 2 。

  • 0 <= prices[i] <= 1000

3. Thinking ​

一种特殊情况即k * 2 > n 则表示可以每天买入或者卖出一次，同股票交易II。

  • f[i][j]表示交易i次，前j天最大利润
  • 集合划分, 第j天是否交易
    • 交易了i次，第j天不交易
    • 交易了i - 1次，第j天交易，枚举买入的时间点

4. Ans ​

class Solution {
public:
    int maxProfit(int k, vector<int>& prices) {
        int n = prices.size();
        if(k * 2 > n){
            // 退化成无限次交易
            int res = 0;
            for(int i = 1; i < n; i ++){
                res += max(0, prices[i] - prices[i - 1]);
            }
            return res;
        }
        vector<vector<int>> f(k + 1, vector<int>(n + 1, 0));
        // f[k][i]表示交易k次，前i天 最大利润
        for(int i = 1; i <= k; i ++){
            for(int j = 1; j < n; j ++){
                // 集合划分，最后第j天是否交易
                // 交易了i次，第j天不交易
                f[i][j] = f[i][j - 1];

                // 交易了i - 1次，第j天交易，枚举买入的时间点
                for(int t = 0; t < j; t ++){
                    f[i][j] = max(f[i][j], f[i - 1][t] + prices[j] - prices[t]);
                }
            }
        }
        return f[k][n - 1];
    }
};

注意这段代码：

for t -> [0, j]:

f[i][j] = max(f[i][j], f[i - 1 & 1][t]  - prices[t] + prices[j]);

f[i - 1 & 1][t] - prices[t]这一块是重复计算了。这里只需要用一个val记录前面的值，当遍历j的时候更新一下val的值就可以了

class Solution {
public:
    int maxProfit(int k, vector<int>& prices) {
        int n = prices.size();
        if(k * 2 > n){
            // 退化成无限次交易
            int res = 0;
            for(int i = 1; i < n; i ++){
                res += max(0, prices[i] - prices[i - 1]);
            }
            return res;
        }
        // 注意到f第一维只和 i - 1有关，所以可以用滚动数组优化空间
        // vector<vector<int>> f(k + 1, vector<int>(n + 1, 0)); 
        vector<vector<int>> f(2, vector<int>(n + 1, 0)); 
        // f[k][i]表示交易k次，前i天 最大利润
        for(int i = 1; i <= k; i ++){
            int val = f[i - 1 & 1][0] - prices[0]; 
            for(int j = 1; j < n; j ++){
                f[i & 1][j] = max(f[i & 1][j - 1], val + prices[j]); 
                val = max(val, f[i - 1 & 1][j] - prices[j]); 
            }
        }
        return f[k][n - 1];
    }
};
